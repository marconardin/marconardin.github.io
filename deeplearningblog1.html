Deep Learning

A subset of machine learning

When writing a program traditionally, the programmer thinks about how to solve a particular problem and manually codes all the steps to solve that problem. Instead in machine learning, the idea is to give an architecture (commonly a neural net in deep learning) inputs and weights that output predictions. Crucially, this model also has the ability to update its weights and make better predictions.

Areas where deep learning is effective

  • Computer Vision (CV)

    • Satellite/drone image interpretation
    • Face recognition
    • Image captioning

  • Radiology

    • Image features (MRI, CT, X-ray)

  • Natural Language Processing (NLP)

    • Speech recognition
    • Classifying documents
    • Finding strings in documents
    • Summarizing documents

  • Games

    • AlphaGo
    • AlphaStar

  • Image generation

    • Colorizing images
    • Increasing image resolution
    • Removing noise from images
